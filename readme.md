# one react template

> a SAO template

## Usage

Install [SAO](https://github.com/egoist/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao or-template my-project
```

### From git

```bash
sao one-react/template my-project
```
